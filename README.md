# Al-Zohaib Store - Premium E-Commerce Website

A professional e-commerce website frontend built with React, Tailwind CSS, and integrated with FakeStore API. Featuring modern UI design with gradient effects, smooth animations, and premium user experience.

## ✨ Features

### 🏠 Home Page
- **Premium Navbar** with gradient logo and smooth hover effects
- **Hero section** with attractive gradient background and dual CTAs
- **Category showcase** with animated cards and hover effects
- **Featured products** with enhanced card design

### 🛍️ Product Listing Page
- **Professional grid layout** with enhanced product cards
- **Advanced filtering** system with emoji icons
- **Real-time search** functionality
- **Product count** display with highlighted numbers

### 🎨 Professional UI Enhancements
- **Gradient backgrounds** throughout the application
- **Smooth animations** and hover effects
- **Enhanced shadows** and border styling
- **Professional color scheme** with indigo/purple gradients
- **Interactive elements** with transform effects

### 📱 Product Detail Page
- **Large product images** with gradient overlays
- **Quantity selector** with enhanced styling
- **Premium button designs** with gradient effects

### 🛒 Shopping Cart System
- **Enhanced cart page** with professional styling
- **Order summary** with modern design
- **Toast notifications** for user feedback

### 🌙 Bonus Features
- **Dark mode** toggle with smooth transitions
- **Loading spinners** with professional styling
- **Responsive design** optimized for all devices

## 🚀 Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start development server:**
   ```bash
   npm run dev
   ```

3. **Open browser** at `http://localhost:5173`

```
src/
```

## 🎨 Design System

### Gradients Used
- **Hero Section**: Indigo → Purple → Pink
- **Buttons**: Indigo → Purple
- **Logo Text**: Indigo → Purple (text gradient)
- **Backgrounds**: White → Gray variations

### Animations
- **Card Hover**: Transform Y (-2px) + Shadow increase
- **Image Hover**: Scale (1.05)
- **Logo Hover**: Scale (1.10)
- **Button Hover**: Shadow effects

### Professional Touches
- **Enhanced shadows** (shadow-lg, shadow-xl)
- **Rounded corners** (rounded-xl)
- **Border styling** with subtle gray tones
- **Backdrop blur** effects on hero section

## 🔧 API Integration

Uses FakeStore API endpoints:
- Products: `https://fakestoreapi.com/products`
- Categories: `https://fakestoreapi.com/products/categories`
- Product Details: `https://fakestoreapi.com/products/:id`
